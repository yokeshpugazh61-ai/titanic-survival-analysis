# Titanic Survival Analysis

## Objective
Perform exploratory data analysis (EDA) on the Titanic dataset to understand factors affecting survival.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Data Cleaning
- Handled missing values:
  - Age → filled with median
  - Embarked → filled with mode
  - Cabin → dropped due to high missing values

## Key Insights
- Females had significantly higher survival (~74%) than males (~19%)
- First-class passengers had higher survival (~63%) compared to third-class (~24%)
- Children had higher survival than adults and seniors
- Survival depends on both gender and passenger class

## Visualizations
- Survival distribution
- Survival by gender
- Survival by passenger class

## Conclusion
Survival on the Titanic was strongly influenced by gender, class, and age.
